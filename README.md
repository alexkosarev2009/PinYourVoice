# Pin Your Voice
> A social network where every place has a voice.

This is my first large-scale Android application, built entirely with **Kotlin** and **Jetpack Compose**.

Pin Your Voice follows **Material 3** guidelines and **Clean Architecture** principles.

The app is still under active development. The latest public release is [available on RuStore](https://www.rustore.ru/catalog/app/com.example.shareyourvoicemapbox)
# Features

## Main Screens

Pin Your Voice introduces a new way to share experiences from places around the world. Instead of plain text reviews user can record a short voice message, attach a photo and place it on the map. Voice reviews capture emotions and authentic experiences that text often cannot.

Users can explore markers both on the interactive 3D globe and through an infinite marker feed. In the feed, user can choose to browse global markers, and friends-only markers.

In Pin Your Voice, you can add friends, create public or friends-only markers, customise your markers and profile, and much more!

<img width="1920" height="1080" alt="PinYourVoice (3)" src="https://github.com/user-attachments/assets/d9926021-d3c5-447d-99b8-75b412f6f199" />

## Dynamic Day / Night cycle

Pin Your Voice tracks the device's local time to decide what map theme to display. There are four different map themes in my app – Dawn, Day, Dusk and Night.

Dynamic time of day and a volumetric environment help immerse the user in the application and bring the map to life.

Additionally, Pin Your Voice supports both light and dark Material 3 themes.

## Clustering

If many markers stack in a same place on the map, they form a **cluster**. This approach improves both performance and map readability.

# Tech Stack
PinYourVoice is built using modern Android development technologies and a Spring Boot backend.
## Android
* Kotlin
* Clean + MVVM Architecture
* Jetpack Compose
* Dagger Hilt Dependency Injection
* Ktor
* ExoPlayer
* Coil
* Mapbox Android SDK
* Amplituda + Waveform

## Backend
* Java
* SpringBoot
* Spring Security
* PostgreSQL + Liquibase DB migration
* Docker
* JWT Authentication
* Amazon AWS S3 API + Yandex Object Storage
